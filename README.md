# Trajectoriya

## Установка и запуск

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/sysol8/trajectoriya.git
   cd trajectoriya
   
2. **Установить зависимости**
    ```bash
   npm i
   
3. **Собрать проект**
   ```bash
   npm run build
   ```

4. **Запустить в превью-режиме**
    ```bash
   npm run preview
   
5. **Перейти на localhost:4173**
